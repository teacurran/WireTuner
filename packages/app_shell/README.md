# app_shell

Flutter UI shell and window management for WireTuner.

## Overview

This package provides the main application shell, window management, and top-level UI structure for the WireTuner vector editor.

## Status

**Iteration I1**: Placeholder package created. Implementation planned for future iterations.

## Planned Features

- Main window widget with menu bar and toolbar
- Document tab management for multi-window support
- Canvas widget integration
- Tool palette and property panels
- Window state management

## Usage

```dart
import 'package:app_shell/app_shell.dart';

const shell = AppShell();
```
